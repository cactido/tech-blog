# tech-blog

## Description
A basic blog project which allows users to register, login/logout, post new blog articles, edit/delete existing articles or comment on existing articles.

## Technology Used
 * Node/Express/JQuery
 * MySQL/Sequelize
 * Bcrypt
 * Handlebars
